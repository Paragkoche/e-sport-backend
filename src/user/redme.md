# USER

## Base URL

```http
http://localhost:8080/user
```

## OPEN ENDPOINTS

- [Regester](regester.md) : `POST /Regester`
- [Login](login.md) : `POST /login`
- [id](id.md) : `GET /:id`
- [usename](username.md) : `GET /username/:username`
- [delete](delete.md) : `DELETE /delete`
